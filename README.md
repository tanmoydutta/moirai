# Moirai
This is an STM32WLE5 standalone module built with Seeed Studio Wio-E5 module. Th objective was to replace the proprietary firmware and make use of the STM32 chip to run a custom firmware which makes use of the built in SubGHz radio for specific purposes. At the back of the PCB there are interface touch pads for Software Debug interface as well as an UART debug connector pads.
